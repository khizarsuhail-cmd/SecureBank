# Secure Banking System

## Overview

This is a Java OOP based banking management system developed using Apache NetBeans.

The project includes:

* Authentication system
* Account management
* Transactions
* Loan management
* Reporting and statements
* Security validation
* Exception handling

---

## Technologies Used

* Java
* OOP
* Java Collections
* Exception Handling
* File I/O
* NetBeans

---

## Features

* Deposit and withdraw
* Fund transfer
* Loan management
* Mini statements
* Full statements
* Admin dashboard
* Audit logging

---

## How To Run

1. Open project in NetBeans
2. Build project
3. Run MainApplication.java
